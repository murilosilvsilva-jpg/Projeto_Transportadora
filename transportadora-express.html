Calculadora de Frete

Função linear: f(x) = ax + b

Modelo de função linear:
f(x) = ax + b

Onde a = custo por km, x = distância e b = taxa fixa de despacho. O custo cresce proporcionalmente à distância.

🧮

Solucionador Algébrico

Bhaskara: ax² + bx + c = 0

Fórmula de Bhaskara:
x = (−b ± √(b²−4ac)) / 2a

• Δ > 0: Duas raízes reais distintas
• Δ = 0: Uma raiz real (dupla)
• Δ < 0: Sem raízes reais

Aplicação: encontrar pontos de equilíbrio em modelos quadráticos de logística.

Como funciona a matemática por trás

Função Linear

Modela custos proporcionais à distância. O coeficiente angular 'a' representa o custo por km e 'b' a taxa fixa de despacho. Quanto maior a distância, maior o custo — crescimento constante.

ax² + bx + c = 0

Equação Quadrática

Utilizada para encontrar pontos de equilíbrio em modelos de custo não-lineares. O discriminante (Δ) determina se existem soluções reais, essencial para análise de viabilidade.
